First steps with raw data from 2025 recording period. 

Merging and initial overview code finds and combines all id.csv files in the directory and make some preliminary summaries to make overview_2025.csv document and cm_2025.csv document. 
It also has a plot for the recording period and missing nights. 

All missing dates are accounted for in the excel on onedrive, no 0-nights have been added back to the dataset.
